clockblock v0.2 arms panel
Prepared by Michael Cousins
michael@wileycousins.com

This panel is 4.8" x 4.2" and divided by v-scoring into 48 0.6" x 0.7" boards. The panel should have 7 evenly-spaced vertical v-scores and 5 evenly-spaced horizontal v-scores. Score lines are indicated in the board outline file.

Requested specs:
  * ENIG finish
  * 0.062", 1 oz copper
  * White soldermask
  * Silkscreen on top, none on bottom
  * Red silkscreen if possible, otherwise black silkscreen

File summary:
Gerbers:
  * clockblock-arms-B_Cu.gbl      - bottom copper
  * clockblock-arms-B_Mask.gbs    - botton soldermask
  * clockblock-arms-Edge_Cuts.gbr - board outline and scoring indication
  * clockblock-arms-F_Cu.gtl      - top copper
  * clockblock-arms-F_Mask.gts    - top soldermask
  * clockblock-arms-F_SilkS.gto   - top silkscreen
Excellon drill files (inches, 2:4 precision, no zero suppression):
  * clockblock-arms.drl - drill file for plated through holes
